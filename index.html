레그 프레스 사용법

레그 프레스는 하체 근력을 효과적으로 강화할 수 있는 대표적인 운동 기구입니다.

올바른 사용 방법

  1. 시트에 앉아 등받이에 등을 완전히 밀착시킵니다.
  2. 발판에 발을 어깨 너비로 벌려 놓습니다.
  3. 안전 잠금장치를 해제합니다.
  4. 무릎을 천천히 구부렸다가 펴는 동작을 반복합니다.
  5. 동작 시 무릎이 발끝을 넘어가지 않도록 주의합니다.

벤치 프레스 사용법

벤치 프레스는 대흉근 발달과 상체 근력 향상에 효과적인 운동입니다.

올바른 사용 방법

  1. 벤치에 누워 양 발을 바닥에 고정합니다.
  2. 바벨을 어깨 너비보다 약간 넓게 잡습니다.
  3. 바벨을 랙에서 들어올립니다.
  4. 천천히 가슴 쪽으로 내렸다가 올립니다.
  5. 호흡을 내쉬면서 바벨을 밀어올립니다.
